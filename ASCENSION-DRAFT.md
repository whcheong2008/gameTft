# Secondary Archetype Assignments — Draft

## Design Principles
1. Secondary should relate to the unit's ability/role thematically
2. Avoid over-clustering on Sage/Mystic/Sorcerer
3. Create interesting team-building options (e.g., a Guardian with Sage secondary lets it count toward healing synergy)
4. No unit gets the same archetype as primary for secondary
