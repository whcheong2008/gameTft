## Evolved Variants

For ANY tier, the evolved version uses **img2img** with the base image as input:

**Echo evolved modifiers** (add to end of base prompt):
```
, more intense, brighter glow, additional energy layers,
more complex flame patterns, powered up, energy overflowing,
secondary color highlights appearing
```
Denoising: 0.4-0.5 (enough to add detail, not enough to lose identity)

**Voidspawn evolved modifiers** (add to end of base prompt):
```
, more developed, additional rocky growths, larger mass,
more active fissures, more sensory pits, additional limb protrusions,
magma flowing more freely, more smoke and ash
```
Denoising: 0.4-0.5

## Differentiation Checklist

Before finalizing any fire unit, verify:
- [ ] Can I tell it apart from other fire units at 64×64 display size?
- [ ] Does its complexity match its tier? (T1 simple, T5 elaborate)
- [ ] Does it clearly belong to the fire element family?
- [ ] Is it NOT humanoid?
- [ ] Does the evolved version look like the same creature "powered up"?
