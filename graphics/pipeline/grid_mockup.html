What you're seeing

This mockup shows how the 256×256 token art will appear on the battle grid at various display sizes. Each token uses CSS gradients as placeholders — in the real game, these would be your AI-generated portraits cropped into circular tokens with element-colored borders.

The fire-echo tokens glow bright with radial light (energy radiating outward). The fire-voidspawn tokens have a darker feel with light trapped inside (heat contained). Each element pair follows this same luminous-echo vs dark-voidspawn pattern.

Tokens include: idle bobbing animation, element glow pulse, health bars, mana bars, and tier star indicators. All driven by CSS — in Unity these would be shader effects and tweens.
